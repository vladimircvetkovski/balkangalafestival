Kick-ass tinyMCE WordPress shortcodes: Button, Icon, Box, Highlight, Divider, Toggle, Google Map, Accordion, Pricing Table, Tabs, and Grid.


== Description ==

Light Shortcode makes you easy to create special contents (Example: tabs, pricing tables, accordion, etc.) without painful of coding manually. The shortcodes can be generated from tinyMCE button. 


= Available Shortcodes =

* Button
* Icon
* Box
* Highlight
* Divider
* Toggle
* Google Map
* Accordion
* Pricing Table
* Tabs
* Grid


<a href="http://www.kharissulistiyono.com/light-shortcode">Live Demo & Documentation</a>
 

== Installation ==

1. Download the Light Shortcode Plugin.
2. In your WordPress Administration, go to Plugins > Add New > Upload, and select the plugin ZIP file.
3. Activate the plugin.
4. From WYSIWYG text editor (Visual), there is a Light Shortcode button where you can select a shortcode item.
5. Some shortcodes have their own attributes, check this guideline on http://www.kharissulistiyono.com/light-shortcode
